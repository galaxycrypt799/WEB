# Coffee Admin

Ứng dụng quản trị cho hệ thống coffee:

- đăng nhập bằng Firebase Auth hoặc local fallback
- xem doanh thu và danh sách đơn
- cập nhật trạng thái đơn
- tạo món mới và upload ảnh lên Firebase Storage

## Chạy app

```powershell
flutter pub get
flutter run
```
